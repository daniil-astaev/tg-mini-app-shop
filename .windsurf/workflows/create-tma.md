# Создание нового Telegram Mini App проекта

## Шаг 1: Подготовка
1. Скопируйте текущий репозиторий в новую папку
2. Откройте в VS Code/Windsurf

## Шаг 2: Настройка бота
1. Создайте нового бота через @BotFather в Telegram
2. Получите токен бота
3. Обновите токен в `bot.js` и переменных окружения

## Шаг 3: Кастомизация приложения
1. Измените дизайн в `src/App.tsx`
2. Обновите товары/услуги в `src/components/`
3. Настройте цвета и стили в `tailwind.config.js`

## Шаг 4: Настройка команд бота
1. Измените команды в `bot.js` под ваши нужды
2. Обновите тексты сообщений
3. Настройте обработку заказов

## Шаг 5: Деплой
1. Запушьте код в GitHub
2. Создайте новый проект в Vercel
3. Настройте переменные окружения
4. Установите webhook для бота

## Шаг 6: Тестирование
1. Протестируйте бота в Telegram
2. Проверьте работу Mini App
3. Убедитесь, что заказы сохраняются

## Быстрый старт командами:
```bash
# Копирование проекта
git clone <original-repo> <new-project-name>
cd <new-project-name>

# Установка зависимостей
npm install

# Локальный запуск
npm run dev
npm run bot  # в отдельном терминале
```
